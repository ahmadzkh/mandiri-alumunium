# Mandiri Alumunium Company Profile

Website company profile premium modern untuk Mandiri Alumunium, usaha aluminium milik Eko Sabari. Project ini diprioritaskan untuk branding, katalog layanan, portfolio foto asli, Local SEO, dan konversi WhatsApp.

## Scope

Project v1.0 hanya company profile static-first.

In scope:

- Homepage premium modern.
- Katalog layanan aluminium.
- Portfolio berbasis foto asli.
- Halaman area layanan Bekasi, Jakarta, dan JABODETABEK.
- CTA WhatsApp sebagai konversi utama.
- Google Maps sebagai CTA sekunder.
- Local SEO, structured data, sitemap, robots, OpenGraph.
- Deployment ke Vercel.

Out of scope:

- Backend.
- Database.
- CMS.
- Ecommerce.
- Checkout.
- Payment gateway.
- Admin dashboard.

## Planned Stack

- Next.js App Router.
- TypeScript.
- TailwindCSS.
- shadcn/ui.
- Vercel.
- Google Analytics.
- Vercel Analytics.
- Google Search Console.
- Playwright.

## Documentation

Implementation must follow these project documents:

- `context.md` — business source of truth.
- `PRD.md` — product requirements and acceptance criteria.
- `DESIGN.md` — brand, visual system, layout, and tokens.
- `CLAUDE.md` — coding and architecture rules.
- `AGENTS.md` — agent workflow and verification policy.
- `TASK.md` — implementation roadmap with task IDs.

## Current Status

Documentation baseline is complete. Source implementation has not started.

## Repository Policy

- Static-first by default.
- No backend/database/CMS for v1.0.
- Portfolio must use original Mandiri Alumunium photos.
- Missing business facts must not be fabricated.
- One logical task equals one commit.
- Conventional commit subject max 72 characters.
